# kata_pub

Pequeño programa para simular el proceso de reserva de un pub.

Desarrolado en php.

Estructura:
.
|- index.php

|- table.php

|- pub.php

|- client.php

Intalación:

Copiar el enlace del repositorio y luego clonarlo:

  git clone https://github.com/rafacode86/kata_pub.git

despues entrar en la carpeta

  cd kata_pub

por ultimo arrancarlo por consola:

  php index.php
